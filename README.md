Programa Registro de Gatos – Kotlin

-Descripción

Este repositorio contiene un programa simple desarrollado en Kotlin que utiliza estructuras de datos del lenguaje.

El programa almacena nombres de gatos en una lista y sus edades en un mapa. Luego muestra la información y calcula el promedio de edad.

-Estructuras utilizadas

MutableList → para guardar los nombres de los gatos.

MutableMap → para asociar cada gato con su edad.

-Objetivo

El objetivo del proyecto es demostrar el uso básico de estructuras de datos en Kotlin como parte de la materia Algoritmos y Estructuras de Datos.
